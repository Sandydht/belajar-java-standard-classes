# System Class
- Class System adalah class yang berisikan banyak utility static method di Java, contoh sebelumnya kita sering menggunakan method println milik field out di class System.
- [Dokumentasi System Class](https://docs.oracle.com/en/java/javase/14/docs/api/java.base/java/lang/System.html)

## Method di System Class
| Method | Keterangan |
|--------|------------|
| String getenv(key) | Mendapatkan environment variable system operasi |
| void exit(status) | Menghentikan program Java |
| long currentTimeMilis() | Mendapatkan waktu saat ini dalam milisecond |
| long nanoTime() | Mendapatkan waktu saat ini dalam nanosecond |
| void gc() | Menjalankan Java garbage collection |
